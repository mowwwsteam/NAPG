Welcome to PhysEd DodgeCoin

Earn fun rewards for staying active! Use DodgeCoins to unlock custom items while getting fit. Scroll down to learn more or use the menu above to jump to specific sections.

What is DodgeCoin?

DodgeCoin is a fictional reward token you earn by completing physical education activities. These tokens can be used in the shop to personalize your profile or website layout.

How It Works

1. Complete activities.
2. Earn DodgeCoins.
3. Spend them in the shop.
4. Stay active and healthy!

Activities

  • 10-minute jog = +5 DodgeCoins
  • 15-minute yoga session = +7 DodgeCoins
  • 20-minute bike ride = +10 DodgeCoins
  • 5 sets of pushups (20 reps) = +6 DodgeCoins
  • 30-minute sports game = +12 DodgeCoins
